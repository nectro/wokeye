Wokeye : A project Management and Monitoring toolkit

Our aim is to make sure that whatever any person works on a project or is leading a project he is able to keep proper track of the work being done, so that a lot of time is saved in continuously taking updates from the members and products are delivered in proper time. Also the Extremely simple user interface makes the application easy to use for users of any page.
